Pantalla per mostrar el log de l'aplicació;.

Autor
oscar

Aquesta pantalla permet:

  • Mostrar tot el log.
  • Exportar el log en format text (.txt)
  • Filtrar els missatges del log de un tipus concret
    • DEBUG.
    • WARNING.
    • ERROR.
  • Mostrar l'últim missatge afegit al log.
